# 部署上线指南 | Deployment Guide > 安全生产专家独立站 — 完整部署步骤 --- ## 第一步：域名注册 ### 推荐域名方案 | 优先级 | 域名 | 优势 | 注册商参考价 | |--------|------|------|------------| | ★★★ | **chinasafetypro.com** | 简短、专业、含义明确，中资安全专家 | ~$12/年 | | ★★★ | **safetychina.com** | 国际视角，适合SEO | 可能已被注册，需查询 | | ★★☆ | **iso45001expert.com** | 精准定位ISO 45001细分领域 | ~$12/年 | | ★★☆ | **seasiasafety.com** | 东南亚安全缩写，区域聚焦 | ~$12/年 | | ★☆☆ | **safetyconsult.asia** | 咨询定位 + .asia后缀 | ~$20/年 | ### 域名注册步骤（以 Cloudflare Registrar 为例，价格最低） 1. 打开 https://dash.cloudflare.com → 注册/登录 2. 点击 **Domain Registration** → **Register Domains** 3. 搜索 `chinasafetypro.com`（或其他选择） 4. 选择年限 → 完成支付 5. **关键DNS配置**（见下方第三步） ### 备选注册商 - **Namecheap** (https://namecheap.com) — 性价比高，$8.88/年起 - **Porkbun** (https://porkbun.com) — 最便宜，$6/年起 - **GoDaddy** (https://godaddy.com) — 最知名，但续费较贵 --- ## 第二步：创建 GitHub 仓库并部署 ### 2.1 创建 GitHub 仓库 1. 登录 https://github.com 2. 点击 **New Repository** 3. 仓库名：`chinasafetypro`（或你的域名） 4. 选择 **Public**（公开，GitHub Pages 才能使用免费版） 5. 勾选 **Add a README file** 6. 点击 **Create repository** ### 2.2 上传文件 方法A：GitHub网页上传（最简单） 1. 进入仓库 → 点击 **Add file** → **Upload files** 2. 把以下文件全部拖拽上传： - `index.html` - `404.html` - `sitemap.xml` - `robots.txt` - `.nojekyll` - `CNAME` - 所有 `程序文件模板_*.html` - 所有其他HTML文件 3. 点击 **Commit changes** 方法B：Git命令行（推荐） ```bash cd your-project-folder git init git add . git commit -m "Initial commit: safety expert website" git branch -M main git remote add origin https://github.com/你的用户名/chinasafetypro.git git push -u origin main ``` ### 2.3 启用 GitHub Pages 1. 进入仓库 → **Settings** → **Pages** 2. Source 选择：**Deploy from a branch** 3. Branch 选择：`main` / `/ (root)` 4. 点击 **Save** 5. 等待1-2分钟，页面会显示： `Your site is live at https://你的用户名.github.io/chinasafetypro/` --- ## 第三步：配置自定义域名 ### 3.1 在 GitHub Pages 设置域名 1. **Settings** → **Pages** → **Custom domain** 2. 输入：`chinasafetypro.com` 3. 勾选 **Enforce HTTPS**（等DNS生效后可勾选） ### 3.2 在域名注册商配置DNS **如果用 Cloudflare：** | Type | Name | Content | Proxy | |------|------|---------|-------| | CNAME | `@` | `你的用户名.github.io` | Off | | CNAME | `www` | `你的用户名.github.io` | Off | **如果用 Namecheap/GoDaddy：** | Type | Host | Value | |------|------|-------| | A | @ | 185.199.108.153 | | A | @ | 185.199.109.153 | | A | @ | 185.199.110.153 | | A | @ | 185.199.111.153 | | CNAME | www | 你的用户名.github.io | ### 3.3 DNS生效等待 - 通常需要 **10分钟 ~ 48小时** - 用 `nslookup chinasafetypro.com` 检查是否生效 - 生效后回到 GitHub Pages 勾选 **Enforce HTTPS** --- ## 第四步：配置支付系统 ### 4.1 Stripe 配置（国际信用卡支付） 1. 注册 Stripe 账号：https://stripe.com 2. 完成商户身份验证（需要护照/营业执照） 3. 获取 **Publishable Key**： - Dashboard → Developers → API keys → Publishable key - 替换 `index.html` 中的 `pk_test_XXXXX` 4. 创建 **Products & Prices**： - Dashboard → Products → Add Product - 创建三个产品：自学版($69)、带教版($179)、企业版($697) - 复制每个产品的 **Price ID**（`price_xxxxx`） - 替换 `index.html` 中 `STRIPE_PRICES` 对象的 `priceId` 5. **简化方案（推荐纯静态站）**： - Dashboard → Payment Links → 创建支付链接 - 直接用 `window.location.href = 'https://buy.stripe.com/xxx'` 跳转，无需后端 ### 4.2 PayPal 配置 1. 注册 PayPal Business：https://www.paypal.com/business 2. 获取 **Client ID**： - Dashboard → Apps & Credentials → REST API apps - 创建App → 复制 Client ID - 替换 `index.html` 中 `YOUR_PAYPAL_CLIENT_ID` 3. 配置 Webhook（可选，用于接收支付通知） ### 4.3 微信/支付宝（可选，需商户资质） 需要中国大陆企业资质： - **微信支付**：https://pay.weixin.qq.com - **支付宝**：https://open.alipay.com - 静态站点方案：使用支付平台的 **H5支付链接** 或 **Native支付二维码** --- ## 第五步：配置邮件订阅 (Mailchimp) ### 5.1 创建 Mailchimp 账号 1. 注册：https://mailchimp.com（免费版支持500订阅者） 2. 完成 onboarding 问卷 ### 5.2 创建 Audience（受众列表） 1. Dashboard → **Audience** → **Create Audience** 2. 填写：Safety Compliance Weekly、公司信息等 3. 进入 Audience → **Signup forms** → **Embedded forms** 4. 点击 **Classic** → 复制 **form action URL** 5. 替换 `index.html` 中两处 `action="https://yourdomain.us1.list-manage.com/subscribe/post?u=XXXXX&id=XXXXX"` 6. 同时替换隐藏字段 `` 中的值 ### 5.3 创建自动化邮件（可选） - 设置 Welcome Email（订阅成功后自动发送） - 设置 Lead Magnet Delivery（下载确认邮件，附上自查清单链接） --- ## 第六步：配置 Google Analytics 1. 登录 https://analytics.google.com 2. 创建 **GA4 Property** 3. 复制 **Measurement ID**（格式：`G-XXXXXXXXXX`） 4. 替换 `index.html` 中两处 `G-XXXXXXXXXX` 5. 在 GA4 中设置转化事件： - `purchase_initiated`：点击支付按钮 - `lead_captured`：提交铅磁表单 - `email_subscribed`：订阅邮件 --- ## 第七步：上线前检查清单 - [ ] 替换所有 `TODO` 标记处的配置值 - [ ] 删除开发者模式黄色提醒条（`#configBanner`） - [ ] 替换所有 `chinasafetypro.com` 为你的真实域名 - [ ] 测试双语切换功能 - [ ] 测试支付弹窗打开/关闭 - [ ] 测试 Mailchimp 表单提交（发送测试订阅） - [ ] 检查移动端响应式布局 - [ ] 使用 Google Search Console 提交 sitemap - [ ] 使用 Google Rich Results Test 验证结构化数据 - [ ] 使用 PageSpeed Insights 检查加载速度 --- ## 文件清单 ``` / ├── index.html # 主页（含支付+订阅+双语+SEO） ├── 404.html # 404错误页 ├── sitemap.xml # 搜索引擎站点地图 ├── robots.txt # 搜索引擎爬虫指令 ├── CNAME # 自定义域名配置 ├── .nojekyll # 禁止Jekyll处理 ├── 东南亚工厂安全合规自查清单50条.html # 铅磁（免费） ├── 东南亚工厂安全合规自查清单100条完整版.html # 付费产品 ├── 模块2_ISO45001核心条款速通_录制脚本+PPT大纲.html # 课程脚本 ├── 30天YouTube内容日历_完整版.html # 内容日历 ├── 30天YouTube内容日历_完整脚本版.html # 内容日历脚本 ├── 程序文件模板_01_危险源辨识与风险评估程序.html # ISO 6.1.2 ├── 程序文件模板_02_法律法规识别与更新程序.html # ISO 6.1.3 ├── ...（共15份模板） └── 程序文件模板_15_管理评审程序.html # ISO 9.3 ```
